## Installation

1. Đảm bảo bạn đã cài đặt docker trên thiết bị của mình.

2. Chạy ứng dụng:
    ```bash
    docker compose up
    ```

3. Mở trình duyệt và truy cập [http://localhost:80](http://localhost:80) để xem ứng dụng.
